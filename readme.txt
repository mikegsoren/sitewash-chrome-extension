Sitewash Installation

1. Unzip files to wherever
2. Go to chrome://extensions in Chrome
3. Check the “Developer mode” checkbox
4. Click “Load unpacked extension”
5. Navigate to the folder where you unzipped the files and click select
6. Make sure “Enabled” checkbox is checked

Usage

1. Click the “SW” icon in the browser toolbar
2. Click “Wash it!” in the popup

Page will return to normal styles on refresh.
